Overview

With this module user can easily return the product of sale order.

Features

It will return product of sale order whose state is in sale and delivery has been made.

Setup & Configuration

Prerequisites

  • No prerequisites are required before the installation.

Configuration

  • No configurations.

Screenshots

Once you have installed the module. Go to Returns Orders module. In that, click Returns Orders menu --> create button. After that, specify the customer name. Based on customer name it will display list of sale order whose state is in sale state as shown in below image. If Delivery Order is not validated for that selected SO then it will raise user error as Please validate the picking of selected sale order and once it is validated you can move further.

Moving on, specify product name, return reason and its return quantity that you want to return.

Once all the details are successfully filled click on Confirm button.

Also at Return Order view level,you can view Delivery OUT Order of selected sale order(SO) in Delivery Orders smart button as shown in below image. Moreover, you can easily view all the returns of that SO in Return Picking smart button as shown in below image.

After that click on Return button and your product will be returned.

Lastly, you can view this return order details in Return Order smart button in Sale Order View as shown in below image.
